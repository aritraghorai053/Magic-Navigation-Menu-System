# ✨ Magic Navigation Menu

A modern and responsive Magic Navigation Menu built using HTML and CSS. The navigation features smooth hover effects, animated active states, and a clean UI design suitable for portfolios, landing pages, and modern websites.

## 🚀 Features

- ✨ Modern and attractive UI
- 📱 Responsive design
- 🎨 Smooth hover animations
- 🌈 Animated active navigation indicator
- 💻 Pure HTML & CSS (No frameworks)
- ⚡ Lightweight and easy to customize

## 🛠️ Technologies Used

- HTML5
- CSS3

1. Clone the repository

```bash
git clone https://github.com/your-username/Magic-Navigation-Menu.git
```
